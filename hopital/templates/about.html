ABOUT

Le motif principal de ce site Web est de fournir un service fiable à nos utilisateurs avec une confidentialité garantie. Ce site Web contient les services du patient pour prendre rendez-vous avec les médecins et peut consulter ses rendez-vous précédents, ordonnances, rapports, etc.

Le patient peut consulter les médecins de sa spécialisation, afin qu’ils puissent prendre rendez-vous avec le médecin respectif de son intrest. Le médecin peut voir tous les détails des rendez-vous afin qu’il puisse donner la prescirption pour les patients. Les données d’ordonnance sont également sauvegardées de sorte que chaque fois que le patient visite à nouveau pour montrer le traitement précédent pris.

La réceptionniste de l’organisation s’occupera des patients externes qui viennent à l’hôpital et maintiendra les paiements pour la consultation hors ligne. L’administrateur peut ajouter de nouveaux médecins, réceptionnistes, afficher tous les rendez-vous, supprimer le compte des médecins, consulter les ordonnances données aux patients par les médecins.
